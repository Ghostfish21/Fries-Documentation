Prefab Parameter

PrefabParameters 为 Unity 预制体（Prefab）在实例化时传入并读取参数提供了一条简洁路径。它通过一个线程安全的 ConcurrentDictionary<int, object[]> 将 GameObject 实例ID 映射到参数数组，并辅以少量扩展方法简化获取。

本组件面向“ 一次性将数据随对象一起带入场景 ”这一高频场景，例如敌人/道具初始化参数、关卡生成配置、动态皮肤/颜色等。

主要特性

  • 一行实例化并传参： gameObject.instantiate(parent, params object[] param)

  • 随处读取： mono.getParams() 或 mono.getParam<T>(index)

  • 线程安全 ：内部使用 ConcurrentDictionary 存储

  • 无需额外组件挂载

快速开始

1）添加命名空间

using Fries;

2）实例化并传参

// 假设 enemyPrefab 是一个可实例化的 GameObject 预制体 var enemy = enemyPrefab.instantiate(parent: someParentTransform, "Goblin", // 名称 5, // 等级 12.5f, // 速度 new Vector3(1,0,0) // 方向 );

3）在脚本中读取

public class Enemy : MonoBehaviour { void Awake() { if (this.hasParam()) { string name = this.getParam<string>(0); int level = this.getParam<int>(1); float speed = this.getParam<float>(2); Vector3 dir = this.getParam<Vector3>(3); // 根据参数做初始化…… } } }

getParam<T>(index) 内部会尝试 Convert.ChangeType 做基础类型转换，并在失败时抛出 InvalidCastException。

工作原理

  1. Awake前： PrefabParameters 在 Awake 前将参数暂存到 paramRegister ，以便构造期可读取。

  2. Awake时 ：试图获取 Params 时， PrefabParameters 会返回 paramRegister 中的内容

  3. Awake后 ：暂存到 paramRegister 中的内容被清理。然后把相同的参数以 GameObject.GetInstanceID 为键写入 ConcurrentDictionary<int, object[]>。

  4. Start时 ：试图获取 Params 时， PrefabParameters 用 GameObject.GetInstanceID() 获取参数列表。

API 参考

instantiate(parent, param)

签名

public static GameObject instantiate(this GameObject gameObject, Transform parent, params object[] param)

参数

  • parent ：新实例的父节点。

  • param ：可选参数列表；将被登记到新实例上以便后续检索。

返回

  • 新创建的 GameObject 实例。

示例

GameObject clone = prefab.instantiate(spawnRoot, 1, "wave-3");

getParams()

签名

public static object[] getParams(this MonoBehaviour mono)

参数

  • （无）

返回

  • 当前 mono.gameObject 的所有参数数组。

示例

object[] all = this.getParams();

getParam (index)

签名

public static T getParam<T>(this MonoBehaviour mono, int index)

参数

  • index ：要获取的参数索引（从 0 开始）。

返回

  • 指定索引处的参数，尝试转换为 T。

异常

  • 当类型不兼容时抛出 InvalidCastException。

示例

int level = this.getParam<int>(0);

hasParam()

签名

public static bool hasParam(this MonoBehaviour mono)

参数

  • （无）

返回

  • 是否为当前对象记录过参数。

示例

if (this.hasParam()) { // do something with parameters }

setParameters(instanceId, param)

签名

public static void setParameters(int instanceId, params object[] param)

参数

  • instanceId ：目标 Unity 对象的 InstanceID（例如 gameObject.GetInstanceID()）。

  • param ：要设置的参数列表；若实例项不存在则会新建。

返回

  • 无。

适用

  • 特殊情况下手动覆写/注入参数。

示例

// 覆写/注入参数 PrefabParameters.setParameters(player.GetInstanceID(), 100, "hard", new Vector3(1, 0, 0));

addParameter(instanceId, param)

签名

public static void addParameter(int instanceId, object[] param)

参数

  • instanceId ：目标 Unity 对象的 InstanceID（例如 component.GetInstanceID()）。

  • param ：要追加的参数数组；若实例项不存在则新建并写入这些参数。

返回

  • 无。

示例

// 追加若干参数（若不存在则新建） object[] extra = { "tag:bonus", 3, true }; PrefabParameters.addParameter(enemy.GetInstanceID(), extra);
